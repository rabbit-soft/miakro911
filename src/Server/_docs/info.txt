Данный сервер выполняет следующие функции
	1. Обновление программного комплекса Miakro 9.11
	2. Сохранение клиентских резервных копий
	3. Хранение информации для Web-отчетов
	4. Удаленное обновление ключей GuardAnt
	
Об обновлении ключей.
	Программа AdminGrd нужна для того чтобы осуществлять добавление новых клиентов, и весити учет ключей Guardant, которые им переданы.
	AdminGrd может:
		-Заводить клиентов
		-Добавлять денег на счет клиента
		-Прошить новый ключ, который находится у разработчика и будет вскоре отправлен клиенту.
		-Назначить прошивку ключу, который находится у клиента. При следующем подключении программы rabDump, на компьютере клиента, к серверу, ключ будет прошит.
	Чтобы прошить ключ сушествует сервис DongleUpdateService который запушен на отдельном компьютере и ждет подключения от Удаленного сервера. 
		К этому компьютеру подключен Guardant-ключ, в котором записаны 2 алгоритма, GSII64 и HASH64. Ключом(16 байт) для этих алгоритмов должен быть ключ TRU.
