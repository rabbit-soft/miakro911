Форма "Отсадка молодняка"

Породаы можете отсадить крольчат к одной из кормилиц, показанных в таблице. При отображении формы Отсадка молодняка, программа заполнит таблицу записями, в которых числятся крольчихи, имеющие подсосных крольчат, разность в возрасте с которыми не превышает значения указанного в Параметры...-Разность объединяемых групп.

Если с крольчхой, от которой вы хотите отсадить крольчат, сидят более чем одна группа молодняка, то перед вами отобразится следующая форма, в которой вам необходимо выбрать, из какой группы будут отсажены крольчата. Выберите нужную группу в раскрывающемся списке и нажмите кнопку Выбрать.

Сверху Формы указан:

  • Кличка подсосных крольчат 
  • Возраст подсосных крольчат
  • Количество доступных для отсадки подсосных крольчат 
  • Порода подсосных крольчат
Таблица содержит следующие колонки:
  • Имя - Здесь указана кличка подходящей кормилицы
  • Возраст - Здесь указан возраст подходящей кормилицы в днях
  • Порода - Здесь указана порода подходящей кормилицы
  • Статус - Здесь указан статус подходящей кормилицы
  • Рейтинг - ;Здесь указан рейтинг подходящей кормилицы
  • Адрес - Здесь указана клетка в которой живет подходящая кормилица
  • Возраст врольчат - Возраст подсосных крольчат сидящих с кормилицей
  • Количество - Количество подсосных крольчат. сидящих с кормилицей
  • Подсажено - Здесь указано количество крольчат. которое вы отсадили данной кормилице из текущей группы молодняка. 
Чтобы отсадить крольчат одной из кормилиц, выделите в таблице нужную запись, после чего будет доступно для редактирования значение в поле Отсадить:  справа от таблицы. Укажите здесь, сколько крольчат подсадить выбранной кормилице. При изменении значения в поле Отсадить;, значение в колонке Подсажено, выбранной записи в таблице, также изменится. При подсадке кроликов одной из кормилиц, значение в поле Количество вверху формы Отсадки молодняка, будет меняться.


Нажмите кнопку Отмена чтобы отменить все текушие измененияи закрыть форму Отсадки молодняка.

При нажатии на кнопку Отсадить, форма Отсадки молодняка закроется, и назаначенным кормилицам отсадится указанное (в поле Подсажено) количество кроликов.
