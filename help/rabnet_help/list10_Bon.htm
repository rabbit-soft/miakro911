Форма "Бонитировка"

Форма Окрола может быть вызвана из Поголовья. 

Здесь указывается классность одного или группы кроликов по следующим признакам: 

  • По весу
  • По телосложению
  • По густоте волосяного покрова
  • По окраске волосяного порова
Классы могут быть следующими (по убыванию):
  • Элита
  • III
  • II
  • I
  • нет - класс не назначен

При нажатии на кнопку Готово, кролику. чье имя указанно сверху формы, будут присвоины выбранный класс по каждому признаку, а также средний класс.

При нажатии на кнопку Отмена форма привоха закроется и все изменения будут потеряны.
