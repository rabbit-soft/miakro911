Основное окно программы

После успешного подключения к Базе Данных (входа). Перед вами появится основное окно программы. Большую ее область занимает панель отображения данных. Отображаемые данные можно менять в помощью кнопок (вкладок):

  • Поголовье - отображает информацию о кроликах, проживающих на ферме.
  • Молодняк - отображает информацию о кроликах, проживающих вместе с матерью
  • Строения - отображает информацию о имеющихся строениях
  • Зоотехплан - отображает работы, которые необходимо провести на ферме

На нижней панели программы (так называемый СтатусБар) отображается информация.

  • Сегодняшнее число
  • Шкала прогресса загрузки данных (ПрогрессБар)
  • Кнопка Обновления данных таблицы
  • Кнопка открывающая окно Фильтров (разное в зависимости от выбранной вкладки)
  • Информация по выбранным строкам и кроликам (разная в зависимости от выбранной вкладки)

Сверху основного окна программы расположено Основное меню программы. В нее входят следующие группы:

  • Ферма
    • Сменить ферму/пользователя ... - При нажатии основное окно программы закроется и перед вами появится форма Входа в систему.
    • Пользователи - Откроет форму Пользователей программы, в которой можно добавить нового или изменить настройки уже имеющихся.
  • Вид
    • Фильтр - Показывает форму фильтра в зависимости от выбранной вкладки в основном меню программы.
    • Параметры - Отображает на экране форму Параметров программы.
    • Архив списаний - Отображает на экране форму архива Списаний, в которой показана информация о всех списанных кроликах.
    • Имена - Отображает на экране Справочник Имен, в которой можно добавить новую или изменить уже имеющиеся записи имен.
    • Породы - Отображает на экране Справочник Пород, в котором можно добавлять новые или изменять старые записи пород.
    • Зоны - Отображает на экране Справочник Зон прибытия, в котором можно добавлять новые или изменять старые записи Местностей.
    • Причины списания - Отображает на экране Cправочник Причин списания, в котором можно добавлять новые или изменять старые записи причин.
    • Учет кормов - Отображает на экране форму Учета кормов. С помощью нее вы можете вести учет за расходом корма.
    • Вакцины - Отображает на экране форму Вакцин. В ней можно добавить Вакцины, которые могут быть вколоты кролику. Так же в этой форме настраивается отображение в зоотехплане.
  • Действия- список зависит от выбранной вкладки Отображения данных
  • Отчеты
    • Состав пород - Вывод на печать Отчет о составе пород.
    • Продуктивность соития - Вывод на печать Отчет о продуктивности соития конкретного самца.
    • Возраст и количество - Вывод на печать Отчет Количества кроликов всех возрастов.
    • Количество по месяцам - Выводит на печать Отчет, в котором указано cколько кроликов имеется на ферме, которые родились в один месяц.
    • Причины списания - Вывод на печать Отчет о причинах списаниях.
    • Списания - Вывод на печать Отчет о списаниях за период времени.
    • Окролы по пользователям - Вывод на печать отчет Окролы по пользователям.
    • Список Случек\Вязок - Вывод на печать отчета о совершенных случках или вязках
  • Настройка
    • Показывать типы ярусов - При включении, в поле адреса будет отображаться описание клетки
    • Показывать типы отделений - писать ли в поле адрес тип МИНИфермы в квадратных скобках
    • Сокращения в таблицах - сокращать ли информацию в полях таблицы Поголовья
    • Сокращение в Зоотехплане - сокращать ли информацию в полях таблицы Зоотехплана
    • Двойные фамилии - Отображать ли в поле Имя фамилию по матери и по отцу
    • Показывать номера - Отображать в фамилиях номер окрола, которым родился кролик
    • Разрешен гетерозис - Назначать ли самкам самцов для спаривания, при соитии с которыми будет допущено Межпородное скрещивание
    • Разрешен инбридинг - Назначать ли самкам самцов для спаривания, при соитии с которыми будет допущено Межродовое скрещивание
    • Параметры - Отображает на экране форму Параметров программы.
  • Справки
    • О программе ... - Отобразит на экране информацию о программе.
    • Помощь - Отобразит на экране это руководство.
