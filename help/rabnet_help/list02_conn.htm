Настройка подключений

После нажатия кнопки Ферма... в окне входа в программу перед вами появится следующее окно:

Здесь отображаются созданные подключения к Базам Данным. Если в поле Название стоит галочка, это значит, что данное подключение будет доступно для выбора при входе в программу. Поле Сохранять пароль показывает, будет ли программа сохранять пароли пользователей. В поле БД написана строка параметров подключения.


Кнопка "Добавить"

Чтобы добавить новое подключение к Базе Данных нажмите кнопку Добавить. После чего появится следующее окно:

Здесь доступны 3 вида действий:

  1. Создать и заполнить БД из файла фермы программы "Миакро 7.2"
  2. Создать БД
  3. Добавить подключение к существующей БД

Чтобы выполнить первые 2 действия, необходимо чтобы в папке с программой находился исполняемый файл - mia_conv.exe

Перед тем как выполнить одно из вышеперечисленных дейсвий, нужно заполнить поля, обьединенные в группу "База данных".

  • Хост - Здесь следует указать IP-адрес или домен, на котором находится БД. Если БД находится на вашем компьютере, то в здесь надо ввести - localhost
  • База Данных - Здесь следует указать название БД, к которой будет данное подключение.
  • Пользователь БД - Здесь следует указать под каким именем, будет осушествляться подключение к БД.
  • Пароль БД - Здесь следует указать пароль пользователя БД (если он имеется).

Если вы хотите выполнить первые 2 действия, то вам нужно заполнить поля, обьединенные в группу Создание БД, чтобы программа могла создать базу данных на компьютере.

  • Администратор - Здесь следует указать имя администратора (как правило это root)
  • Пароль - Здесь следует указать пароль администратора.

1. Создать и заполнить Базу Данных из файла фермы программы "Миакро 7.2"

Нажмите кнопку "Импортировать". Перед вами появится диалоговое окно, в котором нужно выбрать файл с расширением *.mia , из которого вы бы хотели импортировать данные в Базу Данных новой программы.

После выбора файла, перед вами появится окно программы mia_conv.exe, которая приступит к импорту информации из файла фермы (mia-файл) в Базу Данных. Параметры в группе "База Данных" будут копированы из окна "Описание фермы", которое вызвало программу mia_conv.exe. Создание БД из mia-файла может занять довольно продолжительное время время (от 10 минут и больше в зависимости от производительности вашего компьютера). За прогрессом импорта можно следить по шкале в самом низу. После завершения импорта окно программы закроется и в список додключений добавиться запись с данной Базой Данных.


2. Создать БД

Чтобы создать "пустую" Базу Данных. Нажмите кнопку "Создать". После чего на сервере создасться новая БД, в список подключений добавляется новая запись, открывшееся окно закроется.


3. Добавить подключение к существующей БД

Чтобы добавить подключение к уже сушествующей БД, вам требуется ввести в группе параметров База данных такие же параметры, какие вы вводили при создании БД. После чего следует нажать кнопку Сохранить. Окно Описания фермы закроется, и в списов подключений добавится новое.



Кнопка "Изменить"

Чтобы изменить параметры подключения одной из записей, вас нужно выделить ее в списке и нажать кнопку Изменить. Появится уже знакомое окно Описания фермы. Здесь можно изменить нужный вам параметр, после чего нажать кнопку Сохранить. Кнопки Создать и Импортировать при редактировании параметров подключения недоступны.


Кнопка "Удалить"

После нажатия на кнопку Удалить из списка подключений пропадет выделеная запись. БД, к которой было данное подключение останется неизменной.
