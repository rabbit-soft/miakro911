Настройка подключений

После нажатия кнопки Ферма... в окне входа в программу перед вами появится окно, представленное ниже. Здесь можно добавить,изменить или удалить подключения к Базам Данным.

Кнопка "Добавить"

После нажатия на этой кнопки форма изменит свой вид.

Вначале необходимо назначить имя новому подключению в поле Имя фермы

Здесь доступны 2 вида действий:

  1. Создать БД
  2. Добавить подключение к существующей БД

Чтобы выполнить первое действия, необходимо чтобы в папке с программой находился исполняемый файл - mia_conv.exe

Перед тем как выполнить одно из вышеперечисленных действий, нужно заполнить поля, обьединенные в группу "Параметры подключения".

  • Хост - Здесь следует указать IP-адрес или домен, на котором находится БД. Если БД находится на вашем компьютере, то в здесь надо ввести - localhost
  • База Данных - Здесь следует указать название БД, к которой будет данное подключение.
  • Пользователь БД - Здесь следует указать под каким именем, будет осушествляться подключение к БД.
  • Пароль БД - Здесь следует указать пароль пользователя БД (если он имеется).

Если вы хотите выполнить первые 1ое действие, то вам нужно поставить галочку и заполнить поля, обьединенные в группу Создание БД, чтобы программа могла создать базу данных на компьютере.

  • Администратор - Здесь следует указать имя администратора (как правило это root)
  • Пароль - Здесь следует указать пароль администратора.

После того как вы внесли необходимые данные, нажмите кнопку чтобы завершите операцию добавления. Чтобы отменить добавление нажмите кнопку

Имейте в виду, чтобы сохранить внесеные изменения нужно нажать кнопку Сохранить. Если нажать кнопку Отмена, то все ваши действия не проигнорированы.

Кнопка "Изменить"

Чтобы изменить параметры одного из подключений, вам нужно выбрать его в разворачивающемся списке и нажать кнопку с шестеренкой. Появится уже знакомое поле редактирования подключения. Здесь можно изменить нужный вам параметр и Имя фермы, после чего нажать кнопку Готово.

Кнопка "Удалить"

Выберите в разкрывающемся списке Имя фермы подключение, которое хотите удалить. После нажатия на кнопку пропадет выделеная выбранная запись. Имейте в виду, что сама БД на компьютере не удалится. Изменения вступят в силу после нажатия кнопки Сохранить





Создание Базы Данных из файла фермы программы "Миакро 7.2"

Чтобы импортировать в новую программу данные из .mia файла вам понадобиться запустить программу mia_conv.exe из корневого каталога программы. Или же из меню Windows "Пуск"-> Все программы -> Миакро 9.11 -> Импорт из mia

Нажмите кнопку "Импортировать". Перед вами появится диалоговое окно, в котором нужно выбрать файл с расширением *.mia , из которого вы бы хотели импортировать данные в Базу Данных новой программы.

После выбора файла, перед вами появится окно программы mia_conv.exe, которая приступит к импорту информации из файла фермы (mia-файл) в Базу Данных. Параметры в группе "База Данных" будут копированы из окна "Описание фермы", которое вызвало программу mia_conv.exe. Создание БД из mia-файла может занять довольно продолжительное время время (от 10 минут и больше в зависимости от производительности вашего компьютера). За прогрессом импорта можно следить по шкале в самом низу. После завершения импорта окно программы закроется и в список додключений добавиться запись с данной Базой Данных.


2. Создать БД

Чтобы создать "пустую" Базу Данных. Нажмите кнопку "Создать". После чего на сервере создасться новая БД, в список подключений добавляется новая запись, открывшееся окно закроется.


3. Добавить подключение к существующей БД

Чтобы добавить подключение к уже сушествующей БД, вам требуется ввести в группе параметров База данных такие же параметры, какие вы вводили при создании БД. После чего следует нажать кнопкуСохранить. Окно Описания фермы закроется, и в списов подключений добавится новое.
