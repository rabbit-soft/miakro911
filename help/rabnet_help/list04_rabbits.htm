Поголовье

При выборе вкладки "Поголовье" в основном окне программы,перед вами появится информация о кроликах, проживающих на ферме. Панель поголовья разделена на две области: Таблица поголовья, Область отображения деревьев родословной. В Поголовье имеется форма фильтров, с помошью которой можно установить выборку отображаемых записей в таблице поголовья.

Таблица поголовья

Таблица поголовья состоит из следующих колонок:

  • Имя - Здесь отображается имена и фамилии всех кроликов, находящихся на ферме. Кролик может иметь только имя, имя и фамилию, только фамилию. Безымянные кролики помечаются как -0
  • Пол - Здесь указывается пол одного или группы кроликов, сидящих в клетке. Может иметь значения: м(мужской), ж(женский), ? (пол не определен). Также здесь может встретиться значения типа С-# (в клетке сидит сукрольная крольчиха, вместо значения # показан день сукрольности).
  • Возраст - Возраст кролика или группы, сидящих в клетке.
  • Порода - Здесь указана порода одного или группы кроликов, сидящих в клетке. Если включена настройка Параметры...-Сокращенные имена, то порода указывается в сокращенной форме. Названия пород и их сокращенную форму можно проверить\изменить в меню программы Вид-Породы.
  • Вес - Значение последнего взвещивания кролика.
  • Статус - Статус одного или группы кроликов,сидящих в клетке.
  • Пометки - Здесь показаны отметки по кролику: Брак (Б), Готовая продукция (ГП), Прививки (v1,v2,v3).
    Прививки отображаются в формате v#, где #- значение столбца № в Справочнике Вакцин.
    Если отметки находятся в угловых скобках < и >, то это значит, что кролик Готов к реализации.
  • N - Количество кроликов, сидящее в клетке. Может принимать следующие значения: - (в клетке сидит один кролик), [#] (в клетке сидит #-количество кроликов ), +# (вклетке сидит Крольчиха с #-количеством детей).
  • CрВ - Средний возраст подсосных крольчат, сидящих в клетке с крольчихой.
  • Рейтинг - Рейтинг
  • Класс - Класс кролика по результатам бонитировки.
  • Адрес - Адрес клетки в которой проживает один или группа крликов.
  • Заметки - Заметки по данному кролику или группе.
По каждой из этих колонок может проводится сортировка, при нажатии на ее Название. Ширина колонок сохраняется при их изменении.

Действия

При нажатии правой кнопкой мыши на одну или несколько выделенных записей, появляется контекстное меню. В нем, в зависимости от выбранной записи, могут быть следующие пункты:

  • Паспорт - Отображает форму, в которой содержится вся информация по данной записи. Доступна для всех записей. Доступна при выделении нескольких записей.
  • Привоз - Отображает форму Привоза, в котором можно добавить нового кролика(-ов), добавить информацию,посадить его в клетку. Доступна для всех записей.Доступна при выделении нескольких записей.
  • Бонитировка - Отображает форму Бонитировки, в которой присваивается классность одному или группе кроликов. Доступна для всех записей.Доступна при выделении нескольких записей.
  • Списание - Отображает форму Списания, в которой можно списать нужное количество из быбранных клеток. Доступна для всех записей.Доступна при выделении нескольких записей.
  • Отсадить молодняк - Доступно только для записей, в которой числится крольчихи с группой подсосных крольчат.Отображает форму Отсадки молодняка, в которой можно отсадить крольчат к другой крольчихе.
  • Пересадить\Отсадить - Отображает форму Пересадки. В ней можно пересадить кроликов из выранной клетки в свободную, разделить группу подсосных крольчат по половому признаку. Доступна для всех записей. Доступна при выделении нескольких записей.
  • Жилобмен - При выборе 2х записей, откроется окно Пересадок, в котором выбранные записи поменяются адресами.
  • Случка\Вязка - Доступно только для записей, с одной не сукрольной крольчихой, имеющая статус Невеста, Первокролка или Штатная.
  • Прохолостание - Доступно только для записей, с одной сукрольной крольчихой. Отображает форму Прохолостания.
  • Принять окрол - Доступно только для записей, с одной сукрольной крольчихой. Отображает форму Принятия окрола, в которой указывается число родившихся живых (и мертвых) крольчат.
  • Подсчет гнездовых - Доступно только для записей, в которой числится крольчиха с группой подсосных крольчат. Отображает форму Подсчета гнездовых, в которой указывается число крольчат у крольчихи, число умерших и притоптанных.
  • Племенное свидетельство - Доступно для одной выбранной записи. Отобрыжает окно Вывода на печать, в котором содержится информация о классности. родословной. предках данного кроликак или группы.
  • Племенной список - Отобрыжает окно Вывода на печать, в котором содержится информация о классности, породе. Доступна при выделении нескольких записей.
  • Кандидаты на реализацию - Отобрыжает окно Вывода на печать Списка на реализацию.
  • План пересадок - Отобрыжает окно Вывода на печать информацию, необходимую для расселения клетки. Доступна при выделении нескольких записей.
  • Показать родословную - отображает окно Родословной. В нем указано генеалогическое древо выбранной записи.
На нижней панели Поголовья имеются следующие объекты.

  • Текущая дата
  • Шкала прогресса - отображает прогресс загрузки данных с базы данных
  • Кнопка обновления - при нажатии на нее программа обновит данные в таблице поголовья.
  • Кнопка Фильтр поголовья - при нажатии на нее появится форма фильтра, в которой вы можете настроить выборку показываемых записей в таблице поголовья
  • Кнопка Выгрузка в Excel - Создает Excel документ, с содержимым таблицы, отображающей данные.
  • Количество показанных записей в таблице поголовья
  • Общее количество кроликов отображеных в таблице поголовья (без учета бодсосных)
  • Количество выделенных записей и их суммарное количество кроликов


Фильтр Поголовья

Фильтр поголовья является очень полезным инструментом, особенно если с Баззой Данных работают по сети несколько человек. Фильтр позволяет выбрать, кролики с какими признаками будут отображены в таблице поголовья.

Выборка может производиться по следующим параметрам:

  • Порода - выберите в разворачивающемся списке кроликов только какой породы надо отобразить в таблице поголовья. Редактировать список пород можно в меню Вид-Список пород
  • Пол - отметьте галочками, кроликов какого пола нужн отобразить в таблице поголовья. При снятии галочки с поля Самцы или Самки, будут не доступны к редактированию опции фильткаСтатус самца и Статус самки.Если вы уберете галочки с 2х полей, то программа не даст убрать галочку с 3его.
  • Статус самца - отметьте галочками самцов с камим статусом отобразить в таблице поголовья.Если вы уберете галочки с 2х полей, то программа не даст убрать галочку с 3его
  • Статус самки - отметьте галочками самок с камим статусом отобразить в таблице поголовья.Если вы уберете галочки с 3х полей, то программа не даст убрать галочку с 4ого.
  • Дата рождения\Возраст - здесь можно какого возраста отобразить в таблице поголовья. Если с поля От и До сняты галочки, то в таблице поголовья будут отображены кролики всех возрастов.
    Чтобы отобразить в таблице поголовья кроликов, чей возраст больше или равен определенному количеству дней, поставьте галочку в поле От, после чего будут доступны к редактированию поля даты рождения и возраста, находящиеся справа. При изменении поля возраста, в поле даты рождения будет автоматически указана соответствуюая дата рождения, и наоборот.
    Чтобы отобразить в таблице поголовья кроликов, чей возраст меньше или равен определенному количествудней, поставьте галочку в поле До, после чего будут доступны к редактированию поля даты рождения и возраста, находящиеся справа. При изменении поля возраста, в поле даты рождения будет автоматически указана соответствуюая дата рождения, и наоборот. В поле возраста стрелочки вверх и вниз меняют указанное значение на 10, если возраст должен быть не кратный 10, то выделите это поле и впишите небходимое значение самостоятельно с помошью клавиатуры.
  • Вес - здесь можно указать кроликов какого веса отобразить в таблице поголовья. Если с поля От и До сняты галочки, то в таблице поголовья будут отображены кролики с любым весом
  • Плановая работа - ????
  • Сукрольность - Здесь можно настроить отображение в таблице поголовья крольчих, имеющих сукрольность. В разворачивающемся списке доступны 3 типа выборки:
    • Не важно - будут отображены сукрольные и не сукрольные самки
    • Да - будут отображены только сукрольные крольчихи
    • Нет - будут отображены только не сукрольные крольчихи, поля От и До, будут не доступны для редактирования
    Чтобы отобразить в таблице поголовья самок, чья сукрольность больше или равна определенному количеству дней, поставьте галочку в поле От, после чего будут доступны к редактированию поля даты сукрольности и срока, находящиеся справа.
    Чтобы отобразить в таблице поголовья самок, чья сукрольность меньше или равна определенному количеству дней, поставьте галочку в поле До, после чего будут доступны к редактированию поля даты сукрольности и срока, находящиеся справа.
  • Содержит строку - Здесь вы можете указать, строку, которая должна содержаться в имени и фамилии кролика. В основном используется чтобы найти родственников кролика с оределенным именем или фамилией.
Вы можете сохранить настройки фильтров и использовать их в дальнейшем. Чтобы сохранить настройки выборки наберите название фильтра в поле разворачивающегося списка Фильтр (в нижнем правом углу) и нажмите кнопку Сохранить, после чего он будет доступен для выбора в этом разворачивающимся списке.

Нажмите на кнопку Х в верхнем правом углу формы фильтра поголовья, если хотите закрыть окно фильтра.

Нажмите на кнопку Готово если хотите применить данные настройки фильтрации, после чего таблица поголовья отобразит все записи подходящие записи.

Пример работы и сохранения фильтра:Педположим, что нам нужно отобразить в таблице поголовья всех самок, имеющих возраст, от 120 до 200 дней, не сукрольных со статусом "Невеста". Для этого в группе настроек Пол снимаем галочки с полей Самцы и Бесполые. В группе настроек Статус самки снимаем галочки с полей Девочки, Первокролки, Штатные. В группе настроек Датарождения\Возраст ставим галочки в поле От, поле чего в поле возраста набираем число 120, ставим галочки в поле До, поле чего в поле возраста набираем число 200. В группе настроек Сукрольностьвыбирите в разворачивающемся списке Нет. В поле Фильтр наберите с помошью клавиатуры слово "Невесты1" и нажмите кнопку Сохранить, после чего эта запись появится в разворачивающемся списке. Выберите в нем строку "Отчистить", после чего настройки фильтра сбросятся по умолчанию.Выберите в нем строку "Невесты1", после чего появятся настройки которые мы делали ранее. Нажмите на кнопку Готово.

Область отображения деревьев родословной

Область отображения деревьев родословной находится справа от таблицы поголовья, в нем вы можите увидеть генеологическое древо предков кролика, которого вы выделили в таблице поголовья.

Вы можете изменить число отобра; Деревьев родословной по выбранным кроликам в Параметры...-Деревья родословной.
