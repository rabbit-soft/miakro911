Печать "План пересадок"

Печать Плана пересадок может быть вызвана из Поголовья и Молодняка. План пересадок нужен для того чтобы назначать и документировать пересадки из клеток, где сидит крольчиха с подсосными крольчатами. В Плане пересадок указаны: Возраст подсосных крольчат, Адрес, Количество подсосных крольчат. Поле Заметки специально сделано таким большим, чтобы в нем уместилась вся информация о пересадках из текущей клетке. На основе заметок в программе делаются Пересадки.
