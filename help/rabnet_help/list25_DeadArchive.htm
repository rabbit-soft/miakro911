Архив списаний

Форма Архив списаний доступно из Главного меню программы. Здесь показаны все кролики которые были списаны.

Таблица Архива списаний имеет следующие столбцы:

  • Имя кроликов которые были списаны
  • Дата списания кроликов
  • Возраст списанных кроликов
  • Группа - Количество списанных кроликов
  • Порода списанных кроликов
  • Адрес по которому проживали списанные кролики. Если из группы кроликов была списана часть, то в поле Адрес ставится значение "бомж", а в поле Заметки можно посмотреть из какой клетки была списана группа.
  • Причина списания
  • Заметки указанные при списании

При необходиости вы можете востановить списанную запись. Нажмите по нужной записи правой кнопкой мышки и в контекстном меню выберите Восстановить.

Фильтр Архива списаний

В форме Архива списаний так же доступен фильтр. В нем доступны всего 2 опции выборки:

  • Показать списанных кроликов, чье имя содержит строку
  • Какое количество списанных кроликов отобразить в таблице

Внимание:загрузка информации в таблицу списаний очень ресурсоемка. Старайтесь загружать не больше 1000 записей в таблицу. Пользуйтесь фильтром по полю Имя если это действительно необходимо.



Вы можете сохранить настройку выборки с помошью кнопки Сохранить, аналогично как в Фильтре поголовья.

Нажмите на кнопку Готово, чтобы приминить фильтр выборки, либо нажмите на кнопку X в правом верхнем углу формы фильтра чтобы закрыть ее.
