Справочник Вакцин

В справочнике можно добавить Вакцины, которые могут быть вколоты кролику, их продолжительность и возраст, с которого назначать работу по вакцинации в зоотехплане. Прививки можно назначать не только по прошествию Nого количества дней с даты рождения (возраст), но и на Nый день после укола другой прививки.

Таблица имеет следующие столбцы

  • № - Номер Вакцины. В списке поголовья в колонке Пометки вы можете обнаружить значения вида v№. Например: v1,v4,v6. Номер после буквы v соответствует номеру вакциины, которым привит кролик в данный момент.
  • Название Название прививки. Так же отображается в зоотех плане и паспорте кролика.
  • Назначать с... (дней) - День назначения. Если в поле После стоит значение "Рождения", то прививка будет назначена по достижению кроликом возраста N. Если же в поле После стоит имя одной из прививок, то данная прививка будет назначена в зоотехплане по прошествию Nого количества дней от прививки, указнной в поле После.
  • Срок действия (дней) - По истечению данного срока прививка будет назначиена в зоотехплане вновь, если не стоит галочка Назначать единожды
  • После - с какого момента отсчитывать дни, чтобы назначить прививку в зоотехплане.
  • Назначать в ЗооТехПлане - Если данная галочка стоит, то при выборе работ "Прививка" в Зоотехплане, будут показаны кролики, которым нужно вколоть данную вакцину.
  • Назначать единожды - В Зоотехплане, работа по вакцинации данной прививкой будет выводится только один раз для каждого кролика.

В прививках существуют следующие правила

  • По умолчанию в БазуДанных уже добавлена одна вакцина с названием "Прививка" и сроком действия в 365 дней.
  • Чтобы изменить Название, Продолжительность или День назначения кликните 2 раза по одному из нужный полей и введите нужное значение.
  • Чтобы добавить новую прививку нажмите кнопку Добавить новую. Появится новая строка, в которой необходимо внести данные по новой вакцине.
  • Прежде чем добавить новую Вакцину, имейте ввиду, что ее уже нельзя будет удалить.
  • Данные сохраняются автоматически.
  • Номера Вакцин не редактируются.
  • Нельзя назначить прививку после саой себя.
  • При изменении данных в списке Вакцин стоит учитывать, что изменения затронут кроликов которые уже привиты данным видом вакцины.
  • Привить кролика вакциной можно в ЗоотехПлане или в Паспорте.

На изображении выше показана один из возможных наборов прививок. Давайте разберем каждую строчку.

  1. Программа назначит в зоотехплане прививку "Миксоматоз" всем кроликам достигшим возраста в 30 дней. В столбце "Пометки" Поголовья, у данного кролика появится пометка v1. Данная прививка будет действительна 120 дней. Прививка будет назначена только один раз.
  2. Программа назначит в зоотехплане прививку "Гемморогич" всем кроликам привитых вакциной "Миксоматоз" 10 дней назад. В столбце "Пометки" Поголовья, у данного кролика появится пометка v2. Данная прививка будет действительна 120 дней. Прививка будет назначена только один раз.
  3. Программа назначит в зоотехплане прививку "Миксоматоз 2" всем кроликам достигшим возраста в 120 дней. В столбце "Пометки" Поголовья, у данного кролика появится пометка v3. Данная прививка будет действительна 270 дней. По истечению этого срока программа назначит прививку данной вакциной еще раз.
  4. Программа назначит в зоотехплане прививку "Гемморогич 2" всем кроликам привитых вакциной "Миксоматоз 2" 18 дней назад. В столбце "Пометки" Поголовья, у данного кролика появится пометка v4. Данная прививка будет действительна 270 дней. По истечению этого срока программа назначит прививку данной вакциной еще раз.


Спец. уколы

В программе так же предусмотрены специальные прививки.

  • Стимуляция крольчихи - Укол, который действует пару дней. От которого крольчиха более охотно идет на случку\вязку. "Стимулировать" крольчиху можно в Зоотехплане при работах "Случка" и "Вязка". В поле v4 укажите сколько дней действует стимуляция. В поле Назначать на укажите на какой день после стимуляции назначать крольчихе случку\вязку.
