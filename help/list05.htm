Молодняк

При выборе вкладки "Молодняк" в основном окне программы, перед вами появится информация о подсосных\гнездовых кроликах, проживающих на ферме вместе с крольчихой в одной клетке. Панель молодняка разделена на две области: Таблица молодняка, Область отображения деревьев родословной (такое же как в Поголовье).


Таблица молодняка

Таблица молодняка состоит из следующих колонок:

  • Имя - здесь отображена Фамилия молодняка
  • N - количество кроликов в молодняке
  • Возраст - возраст молодняка
  • Пол - здесь может быть 3 доступных значения: м - музской. ж - женский, ? - пол не определен
  • Порода
  • Адрес - в какой клетке числится молодняк
  • Класс - результат определения класности (Бонгитировки)
  • Кормилица - Кличка крольчихи с кем в одной клетке сидит молодняк.
  • Соседи - здесь отображается Фамилия подсаженных крольчат, если они имеются
По каждой из этих колонок может проводится сортировка, при нажатии на ее Название. Ширина колонок сохраняется при их изменении.

Действия

При нажатии правой кнопкой мыши на одну или несколько выделенных записей, появляется контекстное меню. В нем, в зависимости от выбранной записи, могут быть следующие пункты:

  • Отсадить молодняк - Отображает форму Отсадки молодняка, в которой можно подсадить крольчат к другой крольчихе.
  • План пересадок -   Отобрыжает окно Вывода на печать информацию, необходимую для
    расселения клетки. Доступна при выделении нескольких записей.
